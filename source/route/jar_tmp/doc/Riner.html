Riner

Virtuoso Schematic Composer(R)上で動作するピンtoピンのルーティングツールのJAVA外部ルータで用いられるAPI群のjavadocです。 Rinerは、大きくSchematic Composerとの入出力インターフェース部分と外部ルータの2つで構成されます。
  • 設定ファイル・入力ファイル
    • *.prcs
      • 配線に使用するレイヤー層を記述したファイル
    • *.init
      • ルータの設定用のファイル
    • *.riner
      • 接続・セルレイアウト・配線領域が記述されたファイル
      • Schematic ComposerのEdit->riner-routeから範囲を選択した場合に出力される
  • Schematic Composerとのインターフェース
    • riner_util.il out,in,hiで用いられるユーティリティ関数群
    • riner_out.il 外部ルータへの出力のための関数群
    • riner_in.il 外部ルータからの入力をレイアウトに反映させるための関数群
    • riner_hi.il ユーザとの入力と全体の制御を行うための関数群

設定ファイル・入力ファイル

.prcs プロセス記述ファイル プロセスの配線レイヤーの情報を記述するファイル。ルータ起動時に読み込まれる。 ;; sample of *.prcs file (adp ;; プロセス名 3 ;; 配線レイヤー数 1 ;; ピンレイヤー　このレイヤーを接続の始点と終点のレイヤーとして扱う (metal_lgc NONE) ;; (レイヤー名 優先配線方向) (metal_ptl2 HORIZONTAL) (metal_ptl1 VERTICAL) )
  • ';'以降はコメントとして処理される。
  • 全角スペースに注意。
.init ルータ設定ファイル .riner レイアウト・接続情報ファイル ネットの接続情報・バウンダリ・各グリッド座標(x,y)上に存在するセル名が記述されている。 ;; sample of *.riner file ;; net: x1 y1 x2 y2 ネットの接続情報 net: 0 0 1 1 net: 1 2 4 5 ;; boundary: llx lly urx ury 配線領域の左下と右上 ;; grid:よりも先に宣言 boundary: 0 0 100 100 ;; grid: x y cell1 cell2 ... grid: 0 0 jand PTL1 grid: grid:
  • ';'以降はコメントとして処理される。
  • netを記述する位置はファイルのどこでもよい。
  • girdよりもboundaryを先に宣言しておかなければならない。
